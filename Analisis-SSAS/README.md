#  Modelo Multidimensional con SSAS

##  Descripción
Este proyecto consiste en el diseño de un modelo multidimensional orientado al análisis de datos utilizando SSAS.

## Tecnologías utilizadas
- SSAS (SQL Server Analysis Services)
- SQL Server
- 

##  Estructura del modelo

### Tablas de hechos
Contienen los datos principales a analizar.

### Tablas de dimensiones
Permiten segmentar la información (ej: pais, ventas, usuarios).

## Funcionalidad
- Análisis de datos desde múltiples perspectivas
- Organización eficiente para consultas analíticas
- Preparación de datos para BI
