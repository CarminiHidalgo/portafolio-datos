# Pipeline ETL con SSIS

## Descripción
Este proyecto consiste en un proceso ETL (Extract, Transform, Load) donde se integran datos desde archivos CSV hacia una base de datos en SQL Server.

## Tecnologías utilizadas
- SSIS (SQL Server Integration Services)
- SQL Server
- Visual Studio

## Proceso

### 1. Extracción
Se obtienen datos desde archivos CSV.

### 2. Transformación
- Limpieza de datos (valores nulos e incorrectos)
- Estandarización de formatos
- Validación de registros

### 3. Carga
Los datos procesados se insertan en tablas de SQL Server.

## Características
- Validación de calidad de datos
- Manejo básico de errores
- Automatización del flujo ETL
